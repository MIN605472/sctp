Estos datos los hemos generado mediante unos scripts de python para
automatizar el proceso de creación de los casos de prueba.

El script caso_prueba.py crea un caso de prueba, para ejecutarlo se sigue la siguiente gramática:
python caso_prueba.py [N] [M] [C] [A] [B] [output_file]

El script automatiza_pruebas.py ejecuta el script anterior una serie de veces para crear distintas pruebas con
múltiples tamaños del grafo.
