Los ejemplos se han generado con un programa diseñado para este propósito. El programa acepta un número de entrada de intersecciones a generar (N) y la probabilidad de generación de carreteras. Primero, se eligen aleatoriamente una intersección destino y los almacenes. Luego, para cada intersección se genera entre cero y N-1 carreteras con el resto de intersecciones. Después, para cada carretera que sale de una intersección se genera una probabilidad aleatoria entre 0 y 1 y después son normalizadas entre las probabilidades de todas las carreteras que salen de la intersección (salvo para la intersección destino). Finalmente, se traduce al formato especificado y se escribe el fichero.
